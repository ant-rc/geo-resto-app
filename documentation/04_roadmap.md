# GeoResto - Roadmap

## Planning

| Phase | Période | Livrables |
|-------|---------|-----------|
| Cadrage | Nov-Déc 2025 | Specs, maquettes, architecture |
| MVP V1.0 | Jan-Fév 2026 | Recherche géo, filtres, profil |
| V1.5 | Mar-Avr 2026 | Réservation, notifications, favoris |
| V2.0 | Mai-Juin 2026 | Suggestions IA, livraison, espace resto |

## Jalons

- **1 Fév 2026** : MVP fonctionnel
- **29 Mar 2026** : Features complètes
- **19 Juin 2026** : Soutenance PFE

## Stack technique

- **Mobile** : React Native + Expo + TypeScript
- **Backend** : Supabase (Auth, DB, Realtime, Storage)
- **Hosting** : Vercel (API routes si besoin)
- **Maps** : Google Maps API ou Mapbox
- **Paiement** : Stripe

## MVP Scope

1. Auth (email, Google, Apple)
2. Carte restaurants + géolocalisation
3. Filtres : cuisine, prix, distance, allergies
4. Profil utilisateur avec préférences
5. Fiches restaurants détaillées
