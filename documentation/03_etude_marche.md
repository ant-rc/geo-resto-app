# GeoResto - Étude de Marché

## SWOT

| Forces | Faiblesses |
|--------|------------|
| Filtres allergies uniques | Équipe réduite |
| UX centrée utilisateur | Base restos à construire |
| Agrégation livraison | Notoriété zéro |

| Opportunités | Menaces |
|--------------|---------|
| Marché allergies +15%/an | Google Maps pourrait évoluer |
| Restos cherchent visibilité | TheFork domine réservations |
| TheFork peu innovant | Adoption restaurateurs |

## Concurrence

**Google Maps** : Généraliste, pas de filtres avancés
**TheFork** : Réservation seule, peu de personnalisation
**TripAdvisor** : Avis, pas de réservation directe

## Porter - Position

- Pouvoir clients : Élevé (alternatives nombreuses)
- Pouvoir fournisseurs : Moyen (restos dépendants visibilité)
- Menace substituts : Élevée (Google, TheFork)
- Nouveaux entrants : Faible barrière technique
