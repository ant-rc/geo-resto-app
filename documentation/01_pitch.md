# GeoResto - Pitch

## Le Problème

**Utilisateurs :**
- Recherche brouillonne sur Google Maps
- Aucun filtre allergies/régimes
- Pas de notifications promos/events
- Réservation éclatée entre apps

**Restaurateurs :**
- Visibilité limitée vs grandes chaînes
- Difficulté à communiquer promos
- Pas de canal direct clients

## La Solution

| Module | Description |
|--------|-------------|
| Recherche géo | Carte interactive, filtres avancés |
| Profil allergies | Filtrage auto (vegan, halal, sans gluten...) |
| Réservation | 1 clic, style TheFork |
| Notifications | Promos, happy hours, events |
| Livraison | Redirection UberEats, Deliveroo |
| Suggestions IA | Recommandations personnalisées |

## Différenciateurs

| Feature | Google Maps | TheFork | GeoResto |
|---------|-------------|---------|----------|
| Filtres allergies | Non | Basique | Complet |
| Promos/Events | Non | Limité | Oui |
| Suggestions IA | Non | Non | Oui |
| Livraison intégrée | Non | Non | Oui |
